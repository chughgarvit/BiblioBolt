How it works

Three keystrokes to cite

No tab switching. No copy-paste gymnastics. Just type and write.

1

Type \cite

Start typing your citation command in Overleaf. BiblioBolt detects it and opens a popup.

2

Paste or type

Paste a full BibTeX entry from Google Scholar — or just type the citation key if it already exists.

3

Done

The entry goes into your .bib file, \cite{key} appears at your cursor, and Overleaf recompiles.

Features

Built for real paper writing

Every feature comes from actual frustrations writing LaTeX papers.

Multiple .bib files

Working with several bibliography files? Pick the target from a dropdown on every citation, with your last choice remembered.

Duplicate detection

Already added that paper? BiblioBolt checks the full .bib file and skips the insert, still placing the citation key for you.

Key-only mode

Already have the entry in your .bib? Just type the key — no BibTeX paste needed. BiblioBolt inserts \cite{key} instantly.

Batch citations

Paste multiple BibTeX entries at once, or type comma-separated keys. Get \cite{a,b,c} in one action.

Per-project memory

Your .bib file selection is remembered per Overleaf project. Switch projects and your preferences follow.
